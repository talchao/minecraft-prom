Changelog


Latest event:

Minecraft Prom 2026

April 25, 2026


5 Years of Prom!

  • Added birthday cake
  • Added warp compass
    • Fast travel between frequented locations (spawn, games, casino, lake)
    • Alternatively, use /warp <location:optional>
  • Custom fish now properly stack in the inventory when caught
  • Added Pochita plushie
    • Find any of the 5 Pochita plushies around the map for a diamond reward!
  • Added in-game chat announcements
  • Customized pause and load screens:
    • Added logo to pause and load screens
    • Added MHS panorama to load screen
  • Added stop sign block
  • Added Meatball & Ryan Gosling paintings
  • Added speaking head emoji
  • Improved BamBam Hammer bonk behavior
  • Dunk tank now gives players snowballs (prev. dodgeballs)
  • GG emoji no longer automatically inserts into words containing consecutive G's (e.g. rigged).
  • Signs displaying coordinates will teleport player to coordinates when punched
  • Added Mrs. Whiteside NPC

Commands

  • Converted ! -> / commands
    • Allows for tab completion
    • Aligns more with vanilla commands
    • Note: ! commands are still available for now, but will lose support in later events
  • Added /sell – quickly sell specific or all fish
  • Added /back – teleports player to last recorded location
    • Note: Location is currently only recorded after using a custom teleport command (e.g. /warp)

Dodgeball:

  • Lock/unlock player movement
  • Ready-up system
  • Countdown
  • Improve dodgeball pickup behavior

Tic-Tac-Toe: /warp tic-tac-toe

  • Endless mode
  • Custom blocks

Spleef: /warp spleef

  • Reworked user interface for spleef tips
  • Updated TP Stick:
    • Displays time remaining on cooldown when clicked
    • Recharge notification when cooldown is finished
  • Creeper plushie now breaks blocks (prev. only applied knockback)
  • Players no longer take fall damage
  • Improved detection when players exit spleef

Casino: /warp casino

  • Arcade machines can now be found in both the arcade and casino (prev. only arcade)
  • Lootbox adjustments:
    • (+) Dino Domo plushie
    • (–) Frankenstein hat
    • (–) Witch hat
  • Added notification when casino opens
  • Added additional casino entrance

Discord:

  • Added Assocify bot
  • Added @Changelog & @Sneak Peaks roles
  • Updated #FAQ appearance
  • Tweaked role colors

Minecraft Homecoming 2025

Spleef:

  • Spleef arena makeover
  • Added Power-Up cubes (grants player knockback nemo, creeper plushie, or snowballs)
  • Added infographics (click images on wall for quick tips)
  • Supports up to 8 players (previously 6)
  • Other various changes

Loot Boxes: !casino

  • Added Halloween hats (witch & frankenstein) and Luxembourg flag to Loot Boxes
  • Removed college shirts & creeper plushies from Loot Boxes
    • Note: College shirts can still be purchased from Mrs. Ward
  • Loot boxes no longer teleport users with insufficient funds 1000 blocks in the air

Prices:

  • Loot boxes now cost 1 diamond each to open (previously 2)
  • College shirts from Mrs. Ward now cost 1 diamond each (previously 5)
  • Pokemon plushies now cost 5 diamonds each (previously 10)
  • ItemLabel plushies now cost 5 diamonds each (previously 20)

Plushies:

Minecraft Prom 2025

April 26, 2025


Casino: !casino

  • !! NEW: Loot Boxes: Spend 2 diamonds to get a fun, random item.
  • !! NEW: Slot Machine: Bet an amount of diamonds (1, 5, or 10) for a chance to make a profit!

Games:

  • Connect 4: Added outlines to Connect 4 blocks for easier legibility.

General:

  • Court Jesters will now receive Clown Pants alongside the Court Jester Hat.
  • Added more emojis (including rock, paper, & scissors).
